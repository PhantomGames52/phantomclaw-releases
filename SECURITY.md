# Security Policy

The `phantomclaw-releases` repository only hosts compiled release assets.

**Source code security:** Please report vulnerabilities in the source code to the main repository:  
[PhantomGames52/phantomclaw](https://github.com/PhantomGames52/phantomclaw) — see its SECURITY.md.

**Release distribution:** If you suspect a compromised release asset, open an issue in this repository.
